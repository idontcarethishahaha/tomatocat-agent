# Summarize 技能

## 何时使用（触发词）

当用户询问以下内容时立即使用：
- "总结这个链接/文章"
- "这个视频讲了什么"
- "帮我提取这段文本"
- "summarize this"

## 使用方法

1. 使用 `web_fetch` 工具获取网页内容
2. 使用 `read_file` 读取本地文件
3. 将内容整理成简洁的总结

## 输出格式

总结应该包含：
- 核心要点（3-5条）
- 关键数据或结论
- 如果内容很长，先返回摘要，然后询问是否需要展开详细内容

## 注意事项

- 网页内容可能很长，只提取关键信息
- 保持总结的客观性，不要添加个人观点
- 如果遇到付费内容或登录墙，告诉用户无法访问
